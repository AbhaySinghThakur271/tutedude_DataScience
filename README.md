## 📌 Overview

This project analyzes employee data to uncover patterns in **performance** and **attrition**, and builds predictive models using **Machine Learning & Deep Learning**.

It demonstrates how data can be used to make **smart HR decisions** and improve employee retention.

---

## 🎯 Key Features

✔ End-to-end data analysis pipeline
✔ Exploratory Data Analysis (EDA)
✔ Statistical & Probability Analysis
✔ Machine Learning Models
✔ Deep Learning Models
✔ Data Visualization & Insights

---

## 🧠 Problem Statement

Organizations face challenges in:

* Identifying employees likely to leave
* Understanding performance drivers
* Improving retention strategies

👉 This project solves these using **data-driven prediction models**.
